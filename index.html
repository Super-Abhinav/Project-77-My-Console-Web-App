Project on Console

Visual Studio Code is an integrated development environment made by Microsoft for Windows, Linux and macOS.
Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring,
and embedded Git.


The Web Console: Logs information associated with a web page: network requests, JavaScript, CSS, security errors and warnings as well as error, warning and informational messages explicitly logged by JavaScript
code running in the page context. The console is useful for testing purposes. To open the console screen,
right click on the page and select Inspect and then go to console.
